# OperatorKernelO6 — Paper Guide (Consolidated)

Date: 2025-08-16
Branch: guide/consolidated-ssot

## Executive Summary
- Kernel: 6 constructors, 8 unconditional rules (immutable). Public names as in `OperatorKernelO6.Kernel`.
- Termination: per-step hybrid decrease (`HybridDec`); well-founded reverse relations for KO7-safe and MPO-safe subsets.
- Publishable artifacts (Lean names):
  - `MetaSN_Hybrid.hybrid_drop_of_step : Step a b → HybridDec a b`
  - `wf_StepRev_KO7_Safe : WellFounded MetaSN_KO7.SafeStepRev`
  - `wf_StepRev_MPO_Safe : WellFounded MetaSN_MPO.SafeStepMPORev`
- Design choice: avoid asserting a single global measure across all rules; use hybrid certificate and safe subsets to remain sound given κ-duplication and ordinal right-add hazards.

## Kernel Snapshot
- Constructors: `void`, `delta t`, `integrate t`, `merge a b`, `app a b`, `recΔ b s n`, `eqW a b`.
- Step rules (unconditional): `R_int_delta`, `R_merge_void_left`, `R_merge_void_right`, `R_merge_cancel`, `R_rec_zero`, `R_rec_succ`, `R_eq_refl`, `R_eq_diff`.
- Invariants: kernel names/arity unchanged; no axioms/sorry in termination artifacts added here.

## Termination (Hybrid)
- KO7 triple measure: `μ3 : Trace → Nat × (Multiset ℕ × Ordinal)` with outer `deltaFlag` and inner `(κᵐ, μ)`; order `Lex3` is `Prod.Lex (<) LexDM`.
- MPO measure: `ν3m : Trace → Ordinal × (Nat × Nat)`; order `LexNu3m` is `Prod.Lex (<) (Prod.Lex (<) (<))`.
- Certificate:
  - `MetaSN_Hybrid.HybridDec (a b) := LexNu3m (ν3m b) (ν3m a) ∨ Lex3 (μ3 b) (μ3 a)`.
  - `hybrid_drop_of_step` covers every `Step` constructor.
- Safe SN wrappers:
  - KO7-safe: `wf_StepRev_KO7_Safe` (alias of `MetaSN_KO7.wf_SafeStepRev`).
  - MPO-safe: `wf_StepRev_MPO_Safe` (alias of `MetaSN_MPO.wf_SafeStepMPORev`).

## How to Use
- For any single step `h : Step a b`, get a decrease: `MetaSN_Hybrid.hybrid_drop_of_step h`.
- For strategy classes:
  - Use KO7-safe wf when δ/κ side-conditions are enforced (`MetaSN_KO7.SafeStepRev`).
  - Use MPO-safe wf for non-rec rules and `rec_zero` closures.
- See `OperatorKernelO6/Meta/Termination_KO7.lean` for examples at the bottom of the file.

## Soundness Notes
- Ordinal right-addition is not strictly monotone; proofs don’t transport `<` to the left via `… + c` without hypotheses.
- κ-duplication (e.g., `merge_cancel`, `eq_refl`) handled by μ-first MPO; KO7 κ-first used where structurally safe.
